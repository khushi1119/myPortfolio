# 🚀 Khushi Tiwari Portfolio

A modern, responsive, and interactive developer portfolio built with **React.js**, **Tailwind CSS**, and **Framer Motion**. Designed to showcase my projects, technical skills, education, and professional journey through smooth animations, glassmorphism effects, and an engaging user experience.

## 🌐 Live Demo

🔗 **Portfolio Website:** my-portfolio-seven-bice-99.vercel.app

## ✨ Features

* Modern UI with glassmorphism design
* Smooth scroll-based animations
* Interactive project showcase
* Responsive design for all devices
* Animated timeline for Education & Experience
* Dynamic navigation with scroll spy
* Contact form integration using EmailJS
* Clean and scalable component architecture

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

### Navigate to Project

```bash
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
